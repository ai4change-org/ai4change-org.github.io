# AI4Change Website

The official website for the AI4Change movement — hosted on GitHub Pages.

## 🌍 What is AI4Change?

A global open-source initiative where real people use AI as their instrument to solve humanity's most pressing problems. Together. Inspired by [Playing for Change](https://playingforchange.com).
